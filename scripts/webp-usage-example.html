Method 1: Using <picture> Element (Recommended)

This method serves WebP to supported browsers and falls back to JPG for older browsers:

<picture>
  <source srcset="public/Collection/C1/1/1.webp" type="image/webp">
  <img src="public/Collection/C1/1/1.jpg" alt="Description" loading="lazy">
</picture>

Method 2: Simple JPG Usage

Use the optimized JPG files directly:

<img src="public/Collection/C1/1/1.jpg" alt="Description" loading="lazy">

Method 3: Responsive Images

For different screen sizes, you can create multiple versions:

<picture>
  <source media="(max-width: 768px)" srcset="public/Collection/C1/1/1.webp" type="image/webp">
  <source media="(max-width: 768px)" srcset="public/Collection/C1/1/1.jpg">
  <source srcset="public/Collection/C1/1/1.webp" type="image/webp">
  <img src="public/Collection/C1/1/1.jpg" alt="Description" loading="lazy">
</picture>
